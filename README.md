# Pipeline Africa

Landing page for Pipeline Africa — an open-source community helping Africans learn data engineering by building real-world projects with industry experts.

## Stack

Static HTML/CSS site. No build step or dependencies.

- `index.html` — full page markup
- `styles.css` — all styles
- `pa_icon_mark_dark.svg` — logo mark used in the navbar and hero diagram

## Running locally

Open `index.html` directly in a browser, or serve with any static file server:

```bash
npx serve .
```

## Page sections

Hero → Metrics → Features → Mission → Pillars (Join / Build / Impact) → Programs (Workshops, Open Lab, Datasets) → Blog → Join CTA → Footer
